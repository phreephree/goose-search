À propos de Goose

Goose Offers an Openstreetmap Search Engine

Ce site est un moteur de recherche géographique créé par rezemika. Ses résultats sont extraits des données d'OpenStreetMap sous la licence ODbL. Ce site est géré par le framework Django 1.10 avec Python3 et est stylisé avec Bootstrap 3. Il utilise aussi Javascript et jQuery 3 pour l'interactivité.

Son nom est un acronyme récursif : Goose Offers an Openstreetmap Search Engine.

Goose est actuellement en version {{ VERSION.name }}. Voici son dépôt Github : goose-search.

Son logo provient quand à lui de freefavicon.com.

Vie privée

Lors d'une recherche, Goose enregistre certaines informations à des fins statistiques. Ces informations sont : l'objet de la recherche, le rayon de recherche, ainsi que les coordonnées GPS arrondies au degré décimal. Ainsi, il est impossible de connaitre votre position à partir d'une requête faite sur Goose avec une précision meilleure que 130km environ (à la latitude de la France). Enfin, Goose n'utilise aucun outil ou script provenant de Google, Facebook ou Twitter.
